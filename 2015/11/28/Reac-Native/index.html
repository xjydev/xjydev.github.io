React Native

2015-11-28

React Native使你能够在Javascript和React的基础上获得完全一致的开发体验，构建世界一流的原生APP。React Native着力于提高多平台开发的开发效率 —— 仅需学习一次，编写任何平台。(Learn once, write anywhere)。Facebook已经在多项产品中使用了React Native，并且将持续地投入建设React Native。

现在有消息称，Google要让android引入Swift语言，那么Swift语言也是Learn once,write anywhere 了。以后的创业公司移动端就可以招一个工程师了，老板们详细就很省钱啊。但是但是，今年明年这些都不会发生，AppStore上线的应用里面，有多少Swift写的呢，估计还不多。很多人说如果应用重构或者重新开放新项目就用Swift，但是普及还需要时间，API，SDK，完善齐全的开源库。这些都需要用时间、实践去完善。现在我们可以做的就是去不断的深入了解这个语言，并且努力完善它的开发所需。

React这里不多介绍，推荐几个学习React native的好去处。
